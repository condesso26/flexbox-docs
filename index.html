display: flex

A propriedade display: flex deve ser a primeira a ser declarada no CSS. A parti dela é que o CSS vai entender que precisa estilizar o conteúdo do contêiner com a Caixa Flexível (Flexbox).

justify-content

A propriedade justify-content popde ser usada para justificar o conteúdo do elemento contêiner em seu eixo primário, ou seja, eixo x (row, row-reverse) ou eixo y (column, column-reverse).

:center

O valor "center" vai alinhar o conteúdo ao centro do eixo primário.

A propriedade flex-direction é usada no CSS para controlar a direção onde os itens flexíveis serão empilhados dentro de um container flexível. Ela tem quatro valores possíveis: row, row-reverse, column e column-reverse.

A propriedade do CSS align-items define o valor do align-self em todos os filhos diretos como um grupo. No Flexbox, ele controla o alinhamnento dos itens no eixo cruzado. No Laout de grade, ele controla o alinhamnento dos itens no eixo do bloco dentro de sua área de grade.

A propriedade flex-wrap é usada no CSS para controlar como os itens flexíveis dentro de um container flexível são dispostos quando não temos espaço suficiente no contêiner para acomodá-los em uma única linha. Ela afeta o comportamento de quebra de linha dos itens flexíveis.

A propriedade flex-flow é uma junção das propriedades flex-direction e flex-wrap.

align-content é uma propriedade CSS que controla o alinhamento do espaço adicional no eixo de empilhamento de um flex container (eixo principal). Esta propriedade se aplica se espaço adicional estiver disponível após a distribuição de itens flexíveis ao longo deste eixo.
